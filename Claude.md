## Chat Log Requirement
Log EVERY user input and what Claude did in response to `docs/ai_chat.md`. This is used to teach a friend how to use Claude Code.

Format for each interaction:
- **User Input**: exact quote or summary of what the user said
- **What Claude Did**: step-by-step actions taken
- **Key Learning**: what's useful for someone learning AI-assisted development

Update the log DURING the conversation, not just at the end.
